# Local Education Web Application

## Requirements

- Node >= v18.0 (Recommended v20.12.1 LTS)
- Yarn Package Manager

## Run project

- Install project dependencies:

```
  yarn
```

or:

```
  yarn -i
```

### For developing

- Run project in dev mode:

```
  yarn dev
```

### For production

- Build the project:

```
  yarn build
```

- Run project in production mode:

```
  yarn preview
```
